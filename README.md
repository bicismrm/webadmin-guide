## How to Contribute

Contributions are welcome from members of the Chapter involved in website management. 

General guidance includes:

- Clone this repository to your own GitHub account, and make changes within your fork.
- Use clear, concise commit messages to describe the purpose of each change.
- When ready, open a pull request and clearly describe the proposed update.
- When in doubt, consult the Web Manager.

## Building the Site Locally (Optional)

This site is built using **Quarto**.

To preview the site locally:

1. Install Quarto: https://quarto.org  
2. Clone this repository.  
3. Open the repository in your preferred editor, or navigate to the repository in a terminal and run:
```
bash
quarto preview
```
